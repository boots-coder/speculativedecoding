Q2: 为什么天空是蓝色的？
A(8B):
天空呈现蓝色是由于**瑞利散射**现象。太阳光进入地球大气层时，会与空气中的分子（如氮气和氧气）发生散射。波长较短的蓝光比红光更容易被散射，因此蓝光向各个方向散射，使整个天空看起来是蓝色的。

Q6: Explain the difference between supervised and unsupervised learning in one paragraph.
A(8B):
Supervised learning involves training a model using labeled data, where each input is paired with the correct output, allowing the model to learn the relationship between inputs and outputs. Unsupervised learning, on the other hand, uses unlabeled data, and the model identifies patterns or structures within the data without explicit guidance on what to learn.

Q7: Give me a simple Python function that computes Fibonacci numbers iteratively.
A(8B):
Sure! Here's a simple Python function that computes Fibonacci numbers iteratively:

```python
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a
```

This function returns the nth Fibonacci number (0-based index). For example, `fibonacci(5)` returns 5.

Q9: 请用通俗的语言解释大 O 时间复杂度。
A(8B):
大 O 时间复杂度是用来衡量一个算法运行时间随输入数据规模增长而增长的趋势。它描述的是**最坏情况下**，算法的执行时间与输入规模之间的关系。

举个通俗的例子：

- 如果一个算法的时间复杂度是 **O(1)**，表示不管输入多大，它执行的时间几乎不变（比如直接返回一个固定值）。
- 如果是 **O(n)**，表示执行时间随着输入规模 n 线性增长（比如遍历一个数组）。
- 如果是 **O(n²)**，表示执行时间随着输入规模平方增长（比如嵌套循环）。

简单来说，大 O 表示的是：**算法在最坏情况下，会有多“慢”**。它帮助我们选择更高效的算法。
