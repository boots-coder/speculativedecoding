- 公差 $ d = 4 $
- 要求第 10 项，即 $ a_{10} $

$$
a_n = a + (n - 1)d
$$

$$

$$

因为 $ x \ne 2 $，所以 $ x - 2 \ne 0 $，可以约去：

$$
\frac{(x - 2)(x + 2)}{x - 2} = x + 2
$$

---

**答案：**

$$
\boxed{x + 2}
$$

Q2: 为什么天空是蓝色的？
A(4B-Instruct): 天空是蓝色的，是因为阳光穿过地球大气层时，其中的蓝光波长比其他颜色的光更短，更容易被空气中的分子（如氮气和氧气）散射。

这种现象叫做**瑞利散射**（Rayleigh scattering）。根据瑞利散射定律，光的散射强度与波长的四次方成反比，也就是说，波长越短的光（如蓝光、紫光）被散射得越强。

虽然紫光的波长比蓝光更短，理论上应该被散射得更多，但天空看起来是蓝色而不是紫色，原因有两点：

1. **太阳光中蓝光的强度比紫光高**：太阳光中蓝光的成分比紫光多。
2. **人眼对蓝色更敏感**：人眼对蓝色的敏感度高于紫色，所以我们看到的是蓝色。

因此，综合来看，天空呈现蓝色是由于蓝光被大气分子强烈散射，而我们的眼睛接收到的主要是这些散射的蓝光。

✅ 简单总结：  
**天空是蓝色的，是因为阳光中的蓝光被大气中的分子散射到各个方向，我们看到的就是这种散射

Q6: Explain the difference between supervised and unsupervised learning in one paragraph.
A(4B-Instruct): Supervised learning involves training a model on labeled data, where each input is paired with a known output, allowing the model to learn the relationship between inputs and outputs for accurate predictions on new data. In contrast, unsupervised learning uses unlabeled data, where the model identifies patterns, structures, or groupings within the data without any predefined outputs, such as clustering similar data points or reducing dimensionality. While supervised learning is used for tasks like classification and regression, unsupervised learning is often applied for exploratory data analysis, anomaly detection, and data compression.

Q7: Give me a simple Python function that computes Fibonacci numbers iteratively.
A(4B-Instruct): Sure! Here's a simple Python function that computes Fibonacci numbers iteratively:

```python
def fibonacci(n):
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    
    a, b = 0, 1
    for _ in range(2, n + 1):
        a, b = b, a + b
    return b
```

### How it works:
- It returns the `n`-th Fibonacci number.
- Uses two variables to keep track of the previous two numbers.
- Avoids recursion, so it's efficient and fast.

### Example usage:
```python
print(fibonacci(10))  # Output: 55
```
